# Anya Bot

飞书交互命令：

```
hot  fi  it  ai  gh  xhs  mv  vd  push  help
```

| 命令 | 说明 |
|------|------|
| `hot` | 知乎 + 微博热搜 |
| `fi` | 东方财富热榜 |
| `it` | IT之家热榜 |
| `ai` | AI 新闻 (THE DECODER) |
| `gh` | GitHub Trending |
| `xhs` | 小红书 |
| `mv` | 豆瓣电影 |
| `vd` | Bilibili 视频 |
| `push` | 追踪状态概览 |
| `help` | 命令列表 |
