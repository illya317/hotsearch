# 晨间简报输出模板

AI 生成每日简报时，请严格按以下格式输出：

```
🌅 晨间简报 — {{ 日期 }} {{ 星期 }}
━━━━━━━━━━━━━━━━━━━
📌 重点关注
━━━━━━━━━━━━━━━━━━━
1️⃣ [{{ 标签 }}] {{ 标题 }}
{{ 关键数据/指标 }}
{{ 一句话摘要 }}
🔹 原因：
• {{ 要点 1 }}
• {{ 要点 2 }}
• {{ 要点 3 }}
🔹 判断：
{{ 趋势判断/区间预测 }}
━━━━━━━━━━━━━━━━━━━
2️⃣ [{{ 标签 }}] {{ 标题 }}
...
━━━━━━━━━━━━━━━━━━━
📊 其他速览
━━━━━━━━━━━━━━━━━━━
• [{{ 标签 }}] {{ 标题 }} — {{ 一句话 }}
• [{{ 标签 }}] {{ 标题 }} — {{ 一句话 }}
• [{{ 标签 }}] {{ 标题 }} — {{ 一句话 }}
━━━━━━━━━━━━━━━━━━━
```

## 规则

1. **只选值得关注的**：不是每条热榜都列，按主人偏好（preference.md）筛选
2. **标签从 TAG_RULES 里取**：如 `[宏观]` `[AI/科技]` `[健康]` `[地缘政治]` 等
3. **关键数据必须核实**：数字、百分比、日期要准确，不确定就标 "待核实"
4. **判断要留有余地**：用 "预计" "大概率" "区间" 等词，不要绝对化
5. **其他速览 ≤ 5 条**：一句话带过，不展开
